How to set terrain.

Next, the Scene has built in support methods for setting the Terrain for each scene. If no terrain is set, the terrain is set to nullptr for that scene. The string used to identify which terrain to activate is the key used in the LoadResources section.

errcode ExampleScene::Initialize()
{
SetTerrain("Level1");
}
