How to access reverb properties of flimSounds.

Similar to normal properties of the flimSound, each flimSound has a built-in reverb set to inactive by default. This reverb can be activated by adjusting the volume in deciBels. The reverb supports the following adjustments:

Reverb Level: The level in deciBels (-80.0f to 20.0f) of the overall reverb send
Reverb Density: This sets the density of the space; 0 is like a church and 100.0f is like a silo (both reverberant, but shaped differently)
Reverb Predelay: This sets the time in milliseconds before the reverb begins after it receives a dry signal
Reverb time: This sets the time in milliseconds that the reverb will take to drop 60dB (longer reverb times make things more muddy)
Reverb Size: This sets the size of the space, 0 to 1000, and affects the diffusion of the reverb.
errcode Level1::Initialize()
{
music = new flimSound(SoundCallID::LevelMusic);
music->Play(true);
music->SetReverbLevel(-3.0f);
music->SetReverbDensity(100.0f);
music->SetReverbPreDelay(1000.0f);
music->SetReverbSize(100.0f);
music->SetReverbTime(100.0f);
}
